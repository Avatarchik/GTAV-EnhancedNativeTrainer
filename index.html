GTAV-EnhancedNativeTrainer

GTA V PC Enhanced Native Trainer

Welcome to GTAV ENT.

This project is a trainer for GTAV, based on Alexander Blade's ScriptHook and original example, but by now having considerably extended it.

This is intended to be a community project and was formerly open source in public view. Unfortunately this created a number of problems, necessitating a private repository. A number of other individuals and groups decided not to respect the minimal restrictions we had, namely that (a) if you fork the project, change the name, and (b) if you use the code, your work must be open source.

However, it is still very much possible to get involved. If you would like to be granted access, please send an email to gtavent@hushmail.com with a quick note about who you are, why you'd like access, what you plan to contribute etc.

If you do already have access, the project can be found here.

If you don't, you'll get a 404 error.

Apologies for having to provide this check, which is much more annoying for everyone than an open repository.
